# webpack boilerplate

Support: 
- React
- TypeScript
- Ant Design(Or other UI packages you like)
- CSS Module

## Get Started

Install

~~~ bash
yarn
~~~

Start dev server

~~~ bash
yarn dev
~~~

Build

~~~ bash
yarn prod
~~~
